# Accident Detection System using ESP8266

## Aim
To design and develop an IoT based accident detection system that detects vehicle accidents and sends alerts.

## Problem Statement
Road accidents often go unnoticed for a long time, which delays emergency response. This project detects accidents automatically using sensors and sends alerts using ESP8266.

## Components
- ESP8266 (NodeMCU)
- Vibration Sensor
- Buzzer
- LED
- Breadboard
- Jumper Wires

## Software
- Arduino IDE
- Tinkercad / Fritzing

## Project Description
The system detects sudden vibrations indicating a possible accident. When an accident is detected, the ESP8266 triggers an alert through buzzer and LED and can send data through WiFi.
